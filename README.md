# 🚀 Raspidas Workshop

Repositori ini berisi kumpulan kode percobaan dan proyek praktikum menggunakan **Raspberry Pi** dan bahasa pemrograman **Python**. Proyek-proyek ini mencakup kontrol sensor, tombol, motor servo, dan motor DC.

---

## 📂 Daftar Percobaan & File

Berikut adalah daftar eksperimen yang tersedia di dalam repositori ini beserta file kodenya:

| No. | Percobaan / Topik | File Skrip | Deskripsi Singkat |
| :---: | :--- | :--- | :--- |
| **1** | **Kontrol LDR** | `LDR.py` | Membaca nilai sensor cahaya (LDR) menggunakan rangkaian *RC time* pada Raspberry Pi. |
| **2** | **Motor Servo** | `MotorServo.py` | Mengontrol posisi sudut motor servo menggunakan pustaka `gpiozero`. |
| **3** | **Button (Tombol)** | `Button.py` | Membaca status penekanan tombol (*push button*) digital. |
| **4** | **Motor DC** | `MotorDc.py` | Mengontrol arah putaran (maju/mundur) dan kecepatan motor DC menggunakan driver motor. |
